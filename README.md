# консаольная игрушка на 5 минут :)

Консольная игра на C++, где два игрока могут настраивать персонажей, экипировать оружие и сражаться друг с другом.

## Возможности

- Настройка игроков (имя, сброс характеристик)
- 4 вида оружия со случайными характеристиками
- Пошаговая боевая система с критическими ударами
- Обычные и сильные атаки

## Как играть

1. Настрой игроков в меню (опции 1-2)
2. Выбери "Начать битву" (опция 0)
3. Во время боя выбирай тип атаки (1 или 2)
